# Aditya Pareek Portfolio

A modern, premium personal portfolio built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, Lucide icons, and shadcn-style UI primitives.

## Features

- Dark premium glassmorphism interface
- Animated hero with typing text and generated original profile visual
- Sticky responsive navigation
- Scroll reveal animations, cursor glow, particles, scroll progress, and back-to-top control
- Skills, experience, projects, achievements, GitHub stats, resume, and contact sections
- SEO metadata, sitemap, robots file, and optimized image loading
- Server route for downloadable resume PDF

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm start
```

## Deploy

Deploy directly to Vercel. No additional configuration is required.
